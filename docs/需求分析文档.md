## 2. 产品概述

本组件是一个通用的 Markdown 渲染组件，专门针对微信公众号样式排版进行优化。组件以 NPM 包的形式发布，可被其他项目下载使用。

**核心价值**：
- 提供美观的 Markdown 预览效果
- 一键复制即可粘贴到微信公众号
- 支持多主题切换
- 响应式设计适配不同设备

---
