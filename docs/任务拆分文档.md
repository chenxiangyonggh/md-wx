## 2. 项目目标

开发一个可导出的 Markdown 渲染组件，支持：
- **组件导出**：可被其他项目引用使用
- **本地发布**：支持本地直接引用（npm link）
- **NPM 发布**：可通过 npm install 安装使用

## 5. 发布方式

### 5.1 本地发布（npm link）

其他项目可通过以下方式引用：
```bash
# 在组件项目目录
npm link

# 在目标项目目录
npm link @md-wx/react
```

**本地测试说明**：详见 `docs/本地开发指南.md`

### 5.2 NPM 发布

其他项目可通过以下方式安装：
```bash
npm install @md-wx/react
```
