微信公众号 Markdown 渲染组件

这是一个专为微信公众号排版优化而设计的 Markdown 渲染组件，支持多种主题切换、实时预览和一键复制功能。

功能特点

  • 主题切换：支持 5 种预设视觉主题
  • 视图模式：手机 / 桌面两种视图
  • 一键复制：复制渲染后的 HTML 内容
  • 代码高亮：GitHub Dark 风格语法高亮

微信公众号支持有限的 HTML 标签，所有样式必须内联到标签中。推荐使用微软雅黑字体，内容区最大宽度为 677px。
